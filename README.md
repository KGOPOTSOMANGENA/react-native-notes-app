# React Native Notes App

A secure and modern **React Native Notes Application** built with **TypeScript** and **Expo**. Users can create, edit, delete, search, and sort notes by category. Authentication and profile management are included.

---

## **Features**

- User authentication: Register and login
- Notes management: Add, edit, delete, view
- Search notes by keywords
- Sort notes by date (ascending/descending)
- Categories (Work, Study, Personal)
- Profile management: Update username, email, and password
- AsyncStorage used for local database

---

## **Screens**

- **Notes List:** View all notes, search, and sort
- **Add Note:** Create a new note
- **Edit Note:** Update existing note
- **Categories:** Filter notes by category
- **Profile:** View and update user profile
